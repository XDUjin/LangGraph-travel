# 西安景点攻略

## 秦始皇兵马俑博物馆

兵马俑是世界第八大奇迹，展示了秦始皇陵的陪葬坑，被誉为"世界第八大奇迹"。
开放时间：8:30-17:00。
门票价格：120元。
建议游览时间：3-4小时。
交通方式：西安火车站乘坐游5路（306路）直达。

### 参观建议
- 建议请导游讲解，否则只是看泥人
- 一号坑最大最壮观，三号坑是指挥部
- 铜车马展厅是精华，不要错过
- 旺季建议提前网上购票

## 华清宫

华清宫是唐代封建帝王游幸的别宫，因唐玄宗和杨贵妃的爱情故事而闻名。
开放时间：7:00-18:00。
门票价格：120元。
建议游览时间：2-3小时。
交通方式：与兵马俑同线，游5路可达。

### 参观建议
- 《长恨歌》实景演出非常值得观看（需另购票）
- 西安事变旧址也在此处
- 可与兵马俑安排在同一天游览

## 大雁塔

大雁塔是唐代高僧玄奘为保存佛经而修建的佛塔，是西安的标志性建筑。
开放时间：8:00-18:30。
门票价格：40元，登塔另加25元。
建议游览时间：2-3小时。
交通方式：地铁3号线、4号线大雁塔站。

### 参观建议
- 大雁塔北广场有亚洲最大的音乐喷泉（免费）
- 喷泉表演时间：周二至周日12:00、20:30
- 大唐不夜城步行街就在旁边，晚上非常热闹

## 西安城墙

西安城墙是中国现存规模最大、保存最完整的古代城垣。
开放时间：8:00-22:00。
门票价格：54元。
建议游览时间：2-3小时。
交通方式：地铁2号线永宁门站。

### 参观建议
- 建议租自行车骑行城墙，全程约14公里
- 傍晚时分骑行可欣赏日落
- 南门（永宁门）是最主要的城门

## 回民街

回民街是西安著名的美食文化街区，汇聚了各种陕西特色小吃。
开放时间：全天开放。
门票价格：免费。
建议游览时间：2-3小时。
交通方式：地铁2号线钟楼站。

### 参观建议
- 肉夹馍、羊肉泡馍、凉皮是必尝美食
- 建议避开节假日高峰，人非常多
- 大皮院、小皮院等巷子更有当地生活气息
