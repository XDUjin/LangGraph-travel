# 北京景点攻略

## 故宫博物院

故宫博物院是中国最大的古代宫殿建筑群，位于北京市中心，旧称紫禁城。
开放时间：8:30-17:00（周一闭馆，法定节假日除外）。
门票价格：旺季（4月-10月）60元，淡季（11月-3月）40元。
建议游览时间：3-4小时。
交通方式：地铁1号线天安门东站或天安门西站。

### 参观建议
- 建议提前在官网预约门票，旺季需提前7天
- 从午门进入，神武门出，单向游览
- 珍宝馆和钟表馆需单独购票，各10元
- 建议租用语音导览器或跟随导游讲解

## 长城（八达岭）

八达岭长城是长城最具代表性的段落之一，位于北京市延庆区。
开放时间：旺季6:30-19:00，淡季7:00-18:00。
门票价格：旺季40元，淡季35元。
建议游览时间：3-4小时。
交通方式：德胜门乘坐877路公交直达，或乘S2线市郊铁路。

### 参观建议
- 建议早出发避开人流高峰
- 北段较陡但风景好，南段相对平缓
- 建议穿舒适运动鞋，带足饮用水
- 可选择乘坐缆车上下，单程100元

## 颐和园

颐和园是中国现存规模最大、保存最完整的皇家园林。
开放时间：旺季6:30-18:00，淡季7:00-17:00。
门票价格：旺季30元，淡季20元。联票旺季60元，淡季50元。
建议游览时间：3-4小时。
交通方式：地铁4号线北宫门站。

### 参观建议
- 建议购买联票，可参观苏州街、佛香阁等
- 昆明湖游船另收费
- 长廊是必看景点，有精美彩绘

## 天坛公园

天坛是明清两代皇帝祭天祈谷的场所，是中国现存最大的古代祭祀性建筑群。
开放时间：6:00-21:00（祈年殿等景点8:00-17:00）。
门票价格：15元，联票34元。
建议游览时间：2-3小时。
交通方式：地铁5号线天坛东门站。

## 什刹海

什刹海是北京著名的历史文化风景区，包括前海、后海和西海三个湖泊。
开放时间：全天开放（周边酒吧营业至深夜）。
门票价格：免费。
建议游览时间：2-3小时。
交通方式：地铁8号线什刹海站。

### 参观建议
- 夏季可划船，冬季可滑冰
- 周边有很多特色胡同和四合院
- 晚上后海酒吧街氛围很好
- 烟袋斜街是著名的特色商业街
