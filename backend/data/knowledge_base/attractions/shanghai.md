# 上海景点攻略

## 外滩

外滩是上海最具标志性的景点，全长1.5公里，一侧是风格各异的历史建筑群，对岸是陆家嘴现代天际线。
开放时间：全天开放。
门票价格：免费。
建议游览时间：1-2小时。
交通方式：地铁2号线南京东路站。

### 参观建议
- 建议傍晚前往，可同时欣赏日落和夜景
- 夜景灯光秀时间：每日18:00-22:00
- 可乘坐黄浦江游船欣赏两岸风光

## 东方明珠广播电视塔

东方明珠是上海的标志性建筑，高468米，设有观光层和旋转餐厅。
开放时间：9:00-21:00。
门票价格：观光A票120元，观光B票160元。
建议游览时间：2-3小时。
交通方式：地铁2号线陆家嘴站。

## 豫园

豫园是江南古典园林，始建于明代嘉靖年间，距今已有四百多年历史。
开放时间：9:00-16:45（周一闭馆）。
门票价格：旺季40元，淡季30元。
建议游览时间：1-2小时。
交通方式：地铁10号线豫园站。

### 参观建议
- 豫园商城（城隍庙）免费，可品尝上海小吃
- 南翔馒头店的小笼包是必尝美食
- 九曲桥是拍照打卡点

## 田子坊

田子坊是由上海特有的石库门建筑群改建而成的文化创意街区。
开放时间：全天开放（店铺一般10:00-22:00）。
门票价格：免费。
建议游览时间：2-3小时。
交通方式：地铁9号线打浦桥站。

## 陆家嘴

陆家嘴是上海的金融中心，拥有上海中心大厦、环球金融中心、金茂大厦等超高层建筑。
开放时间：各建筑不同。
门票价格：各观景台单独收费，100-200元不等。
建议游览时间：2-3小时。
交通方式：地铁2号线陆家嘴站。

### 参观建议
- 上海中心大厦118层观光厅是目前世界最高的观光层
- 环球金融中心100层观光天阁有透明玻璃地板
- 建议选择晴天前往，视野更好
