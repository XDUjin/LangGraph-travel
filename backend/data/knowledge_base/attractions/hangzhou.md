# 杭州景点攻略

## 西湖

西湖是杭州最著名的景点，也是世界文化遗产，以"西湖十景"闻名于世。
开放时间：全天开放。
门票价格：免费（部分景点单独收费）。
建议游览时间：4-8小时（可分多次游览）。
交通方式：地铁1号线龙翔桥站。

### 参观建议
- 建议租自行车环湖骑行，全程约15公里
- 断桥残雪、三潭印月、雷峰塔是必看景点
- 可乘坐游船游览湖心，船票55元起
- 春天的苏堤和秋天的满觉陇最值得一看

## 灵隐寺

灵隐寺是中国佛教禅宗十大古刹之一，始建于东晋咸和元年。
开放时间：7:00-18:00。
门票价格：飞来峰45元，灵隐寺30元。
建议游览时间：2-3小时。
交通方式：公交7路、807路灵隐站。

### 参观建议
- 需先购买飞来峰景区门票，再购买灵隐寺门票
- 飞来峰石窟造像值得仔细观赏
- 寺内素面馆的素面很有名

## 西溪湿地

西溪湿地是中国首个国家湿地公园，以"一曲溪流一曲烟"著称。
开放时间：8:00-17:30。
门票价格：80元（含电瓶船）。
建议游览时间：3-4小时。
交通方式：地铁5号线西溪湿地北站。

## 千岛湖

千岛湖是世界上岛屿最多的湖泊，水质在中国大江大湖中位居首位。
开放时间：8:00-16:00。
门票价格：150元（含船票）。
建议游览时间：1天。
交通方式：杭州汽车西站乘大巴，约2小时。

### 参观建议
- 建议乘坐游船游览中心湖区
- 梅峰岛是观景最佳点
- 可品尝千岛湖有机鱼头

## 宋城

宋城是以宋代文化为主题的大型旅游景区，《宋城千古情》演出是其最大亮点。
开放时间：10:00-21:00。
门票价格：290元（含千古情演出）。
建议游览时间：4-5小时。
交通方式：公交4路、308路宋城站。

### 参观建议
- 千古情演出场次需提前预约
- 建议下午入园，晚上看完演出再离开
- 园区内有各种宋代主题表演和互动项目
